MxtMenuBar属性(json)
属性 描述
render 显示菜单的位置id[默认是js所在位置] 
var menubar = new MxtMenuBar({'render':'menuDiv','maxMenu':4});
maxMenu 最多显示的菜单个数，超过的放入"更多"中 
同上
MxtMenuBar方法
方法 描述
add(menu) 菜单,参数类型MxtMenu 
var menubar = new MxtMenuBar({'render':'menuDiv','maxMenu':4});

var menu51 = new MxtMenu({'id':'51','name':'菜单一'});
menu51.add(new MxtMenuItem({'id':'2201','name':'item1','action':'http://www.baidu.com','target':'main','icon':'','description':'sssssssssssssss','isShow':true}));

menubar.add(menu51);
show() 渲染菜单 
menubar.show();
MxtMenu属性(json)
属性 描述
id [必须] 菜单id
name [必须] 菜单名称
isShow 菜单是否显示 true or false,默认为true 
var menu51 = new MxtMenu({'id':'51','name':'菜单一','isShow':true});
MxtMenu方法
方法 描述
add(menuItem) 添加子菜单,参数类型MxtMenuItem 
var menu51 = new MxtMenu({'id':'51','name':'菜单一'});
menu51.add(new MxtMenuItem({'id':'2201','name':'item1','action':'http://www.baidu.com','target':'main','icon':'','description':'sssssssssssssss','isShow':true}));
MxtMenuItem属性(json)
属性 描述
id [必须] 菜单id
name [必须] 菜单名称
action 点击菜单动作[待确认？url or 方法]
target 动作响应的页面or框架[待确认]
icon 图标[待确认]
description 描述信息[待确认]
isShow 菜单是否显示 true or false,默认为true 
new MxtMenuItem({'id':'2201','name':'item1','action':'http://www.baidu.com','target':'main','icon':'','description':'sssssssssssssss','isShow':true})
