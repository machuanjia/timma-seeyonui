[1.非阻塞提示]

定义 非打断提示信息。

[2.使用实例]

指定区域
指定区域的提示，
默认居中显示
                    new MxtTip({
                        targetId : "ddd",
                        content : "保持成功！"    
                    });
指定区域 + 偏移
指定区域的提示，
默认居中显示
                    new MxtTip({
                        targetId : "ddd2",
                        offsetTop: 10,
                        content : "保持成功！"
                    });
指定坐标 
                    new MxtTip({
                        top : 460,
                        left : 230,
                        content : "保持成功！"
                    });

[4.参数]

id string id
targetId string 指定区域id
content string 内容
top number position定位top
left number position定位left
keepTime number 持续显示时间
autoShow bool 自动显示
autoClose bool 自动关闭
offsetTop number position定位偏移top
offsetLeft number position定位偏移left
color string 字体颜色
background string 背景颜色
checkPosition bool 验证超出显示区域

[5.方法]

beforeShowCallBack 显示前，回调方法
callBack 关闭后，回调方法
