标准鼠标键操作

操作 效果
Ctrl+A Select all. 全选。
Ctrl+C Copy. 复制。
Ctrl+X Cut. 剪切。
Ctrl+V Paste. 粘贴。
Ctrl+Y Redo. 重做。
Ctrl+Z Undo. 撤消。
Ctrl+F Open the Find dialog box. 打开查找对话框。
Ctrl+P Open the Print dialog box. 打开打印对话框。
Ctrl+N Open a new blank document or the New dialog box. 打开一个新的空白文档或新建对话框。
Tab 切换控件,如:文本框、按钮
Enter 确定
Esc 取消
