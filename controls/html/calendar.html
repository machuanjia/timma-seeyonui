[1.时间控件示例]

定义 可选择不同的时间节点，用于设置时间点及时间段。
示例

显示时间:showsTime:true 
<input id="mycal2" type="text" class="comp" comp="type:'calendar',onUpdate:test,ifFormat:'%Y-%m-%d %H:%M,showsTime:true'"/>




不显示时间: 
<input id="mycal2" type="text" class="comp" comp="type:'calendar',ifFormat:'%Y-%m-%d '"/>




月份选择: 
<input id="mycal3" type="text" class="comp" comp="type:'calendar',ifFormat:'%Y-%m'"/>




调整分钟步长:minuteStep:1 
<input id="mycal4" type="text" class="comp" comp="type:'calendar',ifFormat:'%Y-%m-%d %H:%M,showsTime:true,minuteStep:1'"/>




显示时间:showsTime:true 
<input id="mycal55" type="text" class="comp" comp="type:'calendar',onUpdate:test,ifFormat:'%Y-%m-%d %H:%M,showsTime:true,isOutShow:true'"/>

[4.属性]

属性 类型 描述
displayArea string 要绑定日期控件的ID
ifFormat string 日期选择结果的格式
showsTime boolean 是否显示时间选择
autoShow boolean 是否直接显示时间日历
dateString string 控件打开显示的默认日期
position [] 显示的相对位置[10,10]
onUpdate function 选择日期后执行的方法
