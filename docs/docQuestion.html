7.dialog新增最大化，最小化，关闭触发事件

父页面:
dialog = $.dialog({
...
minParam:{
'show':true,
handler:function(){
var d = dialog.getMin();
}
},
maxParam:{
'show':true,
handler:function(){
var d = dialog.getMax();
}
},
closeParam:{
'show':true,
handler:function(){
var d = dialog.getClose();
},
});


子页面:对应js方法：
function MIN(json){
alert(1)
return json.name
}
function MAX(json){
alert(2)
return json.name
}
function CLOSE(json){
alert(3)
return json.name
}

12.ajaxgrid表格组件的内置排序时跨表问题处理，colModel新增sortname属性用于设置点击列头时自动传入后台进行查询排序时的排序字段名

name : 'orgname',
width : '180',
sortable : true,
sortname : 'o.orgname'
}]
....
});
orgname列被点击时，将使用o.orgname进行排序。
注意：新框架点击列头排序为后台排序，为了性能考虑尽量减少可排序列实现

17.json不要有尾巴','

浏览器兼容：请注意数组的定义，一定不要加个逗号尾巴，如{name: '--------',}，正确写法应为{name: '--------'}，高版本IE没问题，但IE7和其它严格校验的浏览器会出脚本错误。
