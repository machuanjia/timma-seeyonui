CSS3 - 边框

通过 CSS3，能够创建圆角边框，向矩形添加阴影，使用图片来绘制边框。

属性

属性 描述
border-image 设置所有 border-image-* 属性的简写属性。
border-radius 设置所有四个 border-*-radius 属性的简写属性。
box-shadow 向方框添加一个或多个阴影。

示例1 - CSS3 圆角边框

在 CSS2 中添加圆角矩形需要技巧。我们必须为每个圆角使用不同的图片。 在 CSS3 中，创建圆角是非常容易的。 在 CSS3 中，border-radius 属性用于创建圆角：

border-radius 属性允许您向元素添加圆角。
.border_radius{
    text-align:center;
    border:2px solid #a1a1a1;
    padding:10px 40px; 
    background:#dddddd;
    width:350px;
    border-radius:25px;
    -moz-border-radius:25px; /* 老的 Firefox */
}

示例2 - CSS3 边框阴影

在 CSS3 中，box-shadow 用于向方框添加阴影： 


.box_shadow{
    width:300px;
    height:100px;
    background-color:#ff9900;
    -moz-box-shadow: 10px 10px 5px #888888; /* 老的 Firefox */
    box-shadow: 10px 10px 5px #888888;
}

示例3 - CSS3 边框图片

通过 CSS3 的 border-image 属性。 Internet Explorer 不支持 border-image 属性。 border-image 属性规定了用作边框的图片。 您可以使用图片来创建边框：

round:


在这里，图片铺满整个边框。

	.border_round{
		border:15px solid transparent;
		width:300px;
		padding:10px 20px;
		-moz-border-image:url(../images/css_border.png) 30 30 round;	/* Old Firefox */
		-webkit-border-image:url(../images/css_border.png) 30 30 round;	/* Safari and Chrome */
		-o-border-image:url(../images/css_border.png) 30 30 round;		/* Opera */
		border-image:url(../images/css_border.png) 30 30 round;
	}

stretch:


在这里，图片被拉伸以填充该区域。

	.border_stretch{
		border:15px solid transparent;
		width:300px;
		padding:10px 20px;
		-moz-border-image:url(../images/css_border.png) 30 30 stretch;	/* Old Firefox */
		-webkit-border-image:url(../images/css_border.png) 30 30 stretch;	/* Safari and Chrome */
		-o-border-image:url(../images/css_border.png) 30 30 stretch;	/* Opera */
		border-image:url(../images/css_border.png) 30 30 stretch;
	}
